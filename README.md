## Hallucination Evaluation Approach

I classify hallucinations into:

- Factual → incorrect data
- Logical → flawed reasoning
- Contextual → missing dependencies

Evaluation ensures outputs are grounded, complete, and executable.

Includes validation of JSON/API-driven workflows and multi-agent execution pipelines.
